Python Blackjack Game

This is a simple Python command-line application that allows you to play a game of Blackjack against the computer. The game is implemented using Python's built-in random module to simulate shuffling and dealing of cards.

Getting Started To get started with this game, follow these steps:

1. Clone this repository to your local machine. 
2. Start the game by running "python blackjack.py" in your terminal. 
3. Follow the prompts in the terminal to play the game.

How to Play: 
The objective of the game is to get a hand total of 21 or as close to 21 as possible without going over. The game is played against the computer dealer, who follows a set of fixed rules for playing their hand.
